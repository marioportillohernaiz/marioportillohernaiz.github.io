"OTTO" - Voice Recognition AI

Summer 2021

OTTO is a gap year project I worked on where I analysed the different AI technologies in the market. I then created my own voice recognition AI using an open-source hardware platform called "Arduino UNO" and programmed it using the C++ language. This piece of hardware received a voice input through a microphone and outputted a reply through a set of speakers which varied depending on what the input was. Just like the amazon product "Alexa".
